# 🚴 Bike Store Sales Analysis | SQL & Python
SQL and Python project for analyzing bike store sales data using SQLite, Pandas, and data visualization.

# Project Overview
This project analyzes bike store sales data using SQLite, SQL, and Python. The objective is to extract meaningful business insights from customer, product, order, and sales data.

# Business Problem
Retail businesses generate large volumes of transactional data. Without proper analysis, it becomes difficult to identify sales trends, customer behavior, and product performance.This project uses SQL and Python to transform raw sales data into actionable insights.

# Objectives
Analyze customer purchasing behavior
Track sales performance
Identify top-selling products
Evaluate category-wise sales
Perform advanced SQL analysis
Visualize business trends

# Key Insights
Identified high-performing product categories.
Analyzed customer purchasing behavior.
Tracked sales trends across different periods.
Used window functions to calculate rolling averages.
Generated business insights for decision-making.
